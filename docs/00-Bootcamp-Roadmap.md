# Welcome

Welcome to my AI Engineering Bootcamp.

The purpose of this bootcamp is to learn how modern AI systems are built—from the fundamentals to deploying production-ready applications.

This is not just a collection of notes. It is my personal engineering handbook, portfolio, and learning journal.

Every lesson will help me understand not only how to use AI, but also how to build AI-powered applications using open-source technologies.

---

# Goals

By the end of this bootcamp I want to be able to:

- Understand how Large Language Models (LLMs) work
- Run open-source AI models locally
- Build AI-powered web applications
- Build Retrieval-Augmented Generation (RAG) systems
- Build AI automation workflows
- Deploy AI applications on Linux servers
- Understand Docker and containers
- Learn production-ready software engineering practices

# Learning Method

Every lesson follows the same process.

1. Learn the concept
2. Understand why it exists
3. Build a small lab
4. Add the feature to the portfolio project
5. Commit the changes to Git
6. Write a reflection
